RoutePass

Track • Validate • Complete

RoutePass is a checkpoint and event tracking app for organizers and participants. It helps verify checkpoint visits, manage events, and submit event progress with location and optional photo evidence.

What RoutePass does

  • Validates checkpoint visits using your device location
  • Supports event setup and participant management
  • Allows checkpoint submissions with photo and location
  • Uses Google Sign-In for simple account access
